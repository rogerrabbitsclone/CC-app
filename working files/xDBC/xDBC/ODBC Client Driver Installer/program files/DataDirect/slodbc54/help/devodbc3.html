UNIX Compiler Requirements

The ODBC Client has specific compiler requirements on UNIX. Applications must be compiled using the guidelines shown in Table 2-2.

Table 2-2. Compiler Requirements for UNIX
UNIX Platform
Compiler Requirements
AIX reentrant
  • 32-bit compiler version: C Set++ for AIX 3.6.4.0 (using the reentrant version, xlC_r) or higher
  • 64-bit compiler version: C for AIX Compiler 5.0.2.0.
  • 64-bit Compiler flag: ODBC64 (AIX 5.x only)
Solaris
SUNWorkshop (Forte C++) 6.0 or higher
Linux
GNU project C++ Compiler version 2.96
64-bit Compiler flag: ODBC64
HP-UX 11 aCC
HP aC++ Compiler version A.03.05 or higher
