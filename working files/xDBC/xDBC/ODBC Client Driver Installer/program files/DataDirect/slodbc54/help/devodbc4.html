Support for Unicode ODBC W (Wide) Function Calls

The ODBC driver fully supports the SQL-W functions.

On both Windows and UNIX, the ODBC driver automatically determines whether the database is a Unicode database. Table 2-5 shows how the ODBC driver maps the database data types to the Unicode data types.
