SQL Escape Sequences

A number of language features, such as outer joins and scalar function calls, are commonly implemented by DBMSs. The syntax for these features is often DBMS-specific, even when a standard syntax has been defined. The ODBC/JDBC escape sequences contain standard syntaxes for the following language features:

  • Date, time, timestamp, and datetime literals
  • Scalar functions such as numeric, string, and data type conversion functions
  • Outer joins
